Project on console

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control.

Console shows where there can be any errors in your code, and what should be adjusted. In order to open the consoe screen, you would right click on the desired tab/screen, and choose the inspect option. Then, you would look to the top bar and choose console option. To print someting in your console, in your code write, console.log("") and then write the desired message or something else without the brackets.
